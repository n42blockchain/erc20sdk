# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of N42 ERC-20 SDK
- ERC-20 token operations (query, transfer, approve)
- Wallet management (create, import)
- EIP-1559 gas fee support
- WebSocket event subscriptions
- Exponential backoff retry strategy
- Comprehensive error handling
- TypeScript type definitions
- TokenAmount utility for decimal conversion

### Features
- Support for mainnet and testnet
- Automatic gas estimation
- Transaction receipt waiting
- Transfer and Approval event subscriptions
- Network error handling with retries
- Nonce management (auto or manual)
